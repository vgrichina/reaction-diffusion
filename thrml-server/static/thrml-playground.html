What is THRML?

THRML (Thermal) is a probabilistic programming framework built on JAX that uses factor graphs and Gibbs sampling to model complex systems. This demo showcases THRML's ability to maintain unlimited simulation history with full time-travel capabilities.

Key Features:

  • Unlimited History: Every simulation step is stored, enabling perfect time-travel through the entire evolution
  • JAX-Powered: Hardware-accelerated computation using JAX's JIT compilation
  • Factor Graph Model: The Gray-Scott reaction-diffusion system is represented as a probabilistic graphical model
  • Server-Side Computing: Complex simulations run on the server while you interactively explore the results
